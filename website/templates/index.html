Installation instructions
  1. Ensure that you are using the Firefox web browser, if you do not have Firefox installed then download and install it from here.
  2. Download the Firefox extension from here.
  3. Install the extension by dragging the password_advisor.xpi file into the browser. Alternatively open the file from the browser (windows: ctrl + o; mac: command+ o)
  4. When prompted on whether or not to trust the installation, click yes.
  5. Once installed the extension can be accessed using the lock icon at the top right hand corner.
